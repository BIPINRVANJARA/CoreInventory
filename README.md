# 📦 Vyntro - Intelligent Inventory Management System
*A professional team-driven solution for high-performance stock tracking.*

---

Vyntro is a premium, full-stack inventory management solution designed for modern warehouses and retail operations. It features real-time stock tracking, financial reporting, and a high-end user interface.

## 📂 Project Structure

- **`/frontend`**: React + Vite application with premium UI components.
- **`/backend`**: Node.js + Express API handling authentication, inventory logic, and operations.
- **`/database`**: SQL schema and seed data for the system.
- **`/docs`**: Project documentation, including the [Team Collaboration Guide](./docs/team-workflow.md).

## 🚀 Quick Start

### Backend Setup
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`
