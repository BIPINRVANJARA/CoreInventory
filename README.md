# 📦 Vyntro - Intelligent Inventory Management System
*A professional team-driven solution for high-performance stock tracking.*

---

Vyntro is a premium, full-stack inventory management solution designed for modern warehouses and retail operations. It replaces manual registers and Excel sheets with a centralized, real-time inventory platform.

## 🎯 Problem Statement
Build a modular IMS that eliminates manual error and provides warehouse owners with a high-fidelity view of their stock, movements, and financial metrics.

## 🛠️ Tech Stack
- **Frontend**: React + Vite (Vanilla CSS)
- **Backend**: Node.js + Express
- **Database**: MySQL / PostgreSQL
- **Design**: Glassmorphism & Premium UI (Stitch Architected)

## 📂 Project Structure
- **`/client`**: React application with premium UI components and `PremiumSelect` architecture.
- **`/server`**: Node.js API handling authentication, inventory logic, and real-time ledger tracking.
- **`/database`**: Universal SQL schema and seed data.
- **`/docs`**: Project documentation and [Team Collaboration Guide](./docs/team-workflow.md).

## 🚀 Quick Start

### Backend Setup
1. `cd server`
2. `npm install`
3. `npm start`

### Frontend Setup
1. `cd client`
2. `npm install`
3. `npm run dev`
