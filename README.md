## Problem Statement
Build a modular IMS that replaces manual registers and Excel sheets 
with a centralized, real-time inventory platform.

## Tech Stack
- Frontend: React + Tailwind CSS
- Built with: Antigravity / Stitch

## Features
- Dashboard with KPIs
- Product Management
- Receipts & Delivery Orders
- Internal Transfers
- Stock Adjustments
- Move History / Stock Ledger
- Warehouse Management
